# 数据 内存 变量

数据 是存储在内存里面的东西

内存 是内存条通电后可存储数据的空间  内存有2个数据 1，内存存储的数据 2，地址值
内存分类 1，栈 声明变量   2，堆  （对象）
硬盘是永久保存
变量 可以变化的量 组成 变量名和变量值组成 var a（变量名）=18（变量值）   变量名是标识作用  变量值是内存中存储的数据
常量 不可以变化的量
